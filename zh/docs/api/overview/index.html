API 概述

本节提供 Contexify API 的参考，每个类、方法和接口都有详细的文档和示例，帮助您了解如何有效地使用它们。

核心类​

Contexify 围绕几个核心类构建，这些类为依赖注入提供了基础：

  • Context - 绑定和依赖项的中央注册表
  • Binding - 表示键和值之间的连接
  • ContextView - 跟踪匹配特定过滤器的一组绑定

装饰器​

Contexify 提供了一组装饰器，使在 TypeScript 中使用依赖注入变得容易：

  • @injectable() - 将类标记为可注入
  • @inject() - 通过绑定键注入依赖项
  • @inject.tag() - 注入匹配特定标签的所有依赖项
  • @inject.getter() - 注入一个可以稍后用于获取依赖项的函数
  • @inject.view() - 注入一个跟踪匹配过滤器的绑定的 ContextView
  • @config() - 为当前绑定注入配置
  • @intercept() - 将拦截器应用于方法或类

接口和类型​

Contexify 定义了几个在整个框架中使用的接口和类型：

  • Provider - 定义一个可以动态创建值的类
  • Interceptor - 定义一个可以拦截方法调用的类
  • ContextObserver - 定义一个可以观察上下文事件的类
  • InvocationContext - 提供有关方法调用的信息
  • BindingScope - 定义绑定的作用域
  • Constructor - 表示类构造函数
  • ValueOrPromise - 表示值或值的 Promise
  • Getter - 表示返回值或值的 Promise 的函数
  • BindingFilter - 表示过滤绑定的函数
  • BindingComparator - 表示比较绑定的函数
